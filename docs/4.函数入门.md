## 函数入门



### 什么是函数

函数指的是一个包裹了一些代码的代码块，通过调用这个函数可以执行里面的代码；

作用：

- 复用代码：将一些通用代码封装到函数中，方便复用
- 结构清晰：当某个功能需要些很多代码的时候，我们可以拆分为多个函数，代码看起来就会更加清晰



### 函数的基本使用



定义函数：

````python
def 函数名():
    函数体
    函数体
````

- def：定义函数的关键字
- 函数名()：括号中可以携带参数
- 函数体：具体执行的代码



调用函数：

````python
函数名() # 直接通过函数名()调用，如果有参数则需要传参数
````



### 参数



#### 什么是参数



参数指的是传给函数的数据；参数只有本函数中可以调用



参数具体分为入参，形参；

**入参**（实参）：调用函数时传的参数（**由调用者决定，传几个则是由函数预先定义了几个决定，数量必须保持一致,定义了默认值参数除外**）

**形参**：定义函数时预先设置好的参数（参数的值具体是什么由调用函数时传什么决定，实参的使用范围仅限于本函数；）



####  **入参的使用**

1. **位置参数**：调用函数时，传入的参数的位置决定了传递给函数那个形参
2. **关键字参数**：调用函数时通过 `形参名 = 值` 的形式去传参，与顺序无关（可以和位置参数混用，但是第一个参数必须是位置参数）



````python
def test(var1, var2):
    print(var1)
    print(var2)

test(1,2) # 位置参数
test(var2 = 2, var1 = 1) # 关键字参数
test(1, var2 = 2) # 混用
tes(var1 = 1, 2) # 报错，混用情况下不允许第一个是关键字参数
````



#### **形参的使用**

1. **限制传参方式**：`/`前面只能用『位置参数』，`*`后面只能用『关键字参数』
2. **参数默认值**：可以提前设置默认值，**如果调用时没传参数，则使用默认值**，否则使用传入的参数（注意：**默认参数后面的参数也必须写默认值**，否则报错）
3. **可变参数**：某些情况下不确定会传入多少个参数，或者是否传参数；这种情况可以用可变参数（注意：可以混用可变位置参数和可变关键字参数，但是需要第一个是可变位置参数）
   1. 方式一：`*形参名`，多个位置的参数最终会打包为元组（只能传位置参数）
   2. 方式二：`**形参名`，多个位置的参数最终会打包为字典（只能传关键字参数）



````python
# 限制传参方式
def test(var1, /, var2, *, var3):
    prtin(var1,var2, var3)
test(1, 2, var3 = 3) # ok
test(1, var2 = 2, var3 = 3) # ok
test(var1 = 1, var2 = 2, var3 = 3) # 错误

# 参数默认值
def run(var1, var2, var3 = 100): # ok
    prtin(var1,var2, var3)
def run2(var1, var2 = 100, var3): # 错误 var3
    prtin(var1,var2, var3)
run(1,2,3) # ok
run(1,2) # ok

# 可变参数
def success(*args):
    print(args)
def success2(**kwargs):
    print(kwargs)
success(1,2,3)
success2(a = 1, b = 2, c = 3)

# 混用可变参数
def task(a,b,*args, d = 10, **kwargs):
    print(a)
    print(b)
    print(args)
    print(d)
    print(kwargs)

````



#### 特殊字面量None

None的意思是空值，无值；



关键点：

1. `None` 出现在布尔判断中，会被当作false处理
2. 不能参与数学计算，也不能和字符串拼接
3. 不给函数设置返回值，则函数就会默认返回`None`



某些情况，我们想定义某个变量，但是不确定他的值，可以用None



````python
msg = None
msg = "error!"
type(msg) # NoneType
````



### 返回值



指函数将数据返回给调用者；关键字：`return`，所有函数都有返回值，即使不去设置也会默认返回`None` 

return的作用：

1. 结束函数运行
2. 返回数据给调用者



````python
def task():
    ab = 100
    return ab
arg = task()
print(arg) # 100
````



### 作用域

作用域指的是变量起作用的范围，哪里可以用，哪里不能用；

全局变量随程序的开始而创建，程序结束后销毁

局部变量随函数的调用而创建，调用完毕销毁

#### **全局作用域**

- 整个`.py`文件最外层的位置，也就是函数外层的范围
- 定义在全局作用域的变量在整个程序中都可以访问



````python
sex = 1 # 全局变量
def test():
    num = 100 # 局部变量
arg = 200 # 全局变量
````



#### **局部作用域**

- 函数内的范围就是局部作用域
- 只有当前函数可以访问这个变量



````python
sex = 1 # 全局变量
def test():
    num = 100 # 局部变量
arg = 200 # 全局变量
````



#### global关键字

global在函数内使用可以声明变量为全局变量

注意：

1. 局部变量和全局变量名称一样时，不能用global修饰局部变量
2. 不用global修饰变量也可以使用全局变量，只不过不能修改全局变量的值



````python
a = 100

def test():
    global a  # 使用 global 关键字，将a声明为全局变量。
    a = 300
    print('函数中的打印（a）', a)
test()
print('全局的打印（a）', a)
````



### 递归调用函数



递归调用函数指自己调用自己

递归的时候一定要小心，防止无限递归，也就是死循环，因此一定要有结束条件可以终止递归



````python
def test(num):
    print("test" + str(num))
    if num == 100: # 结束递归的条件
        return
    else:
        test(num + 1)

test(1)
````
