## python基础

### 变量与常量

#### 变量
变量是程序中用来存储数据的容器，可以在程序运行过程中改变其的值。

变量的命名规则如下：
- 变量名必须以字母（a-z或A-Z）或下划线（_）开头，后面可以跟字母、数字（0-9）或下划线。
- 变量名不能是Python的保留字（如if、else、for等）。
- 变量名区分大小写。

python中变量的创建和赋值是同时完成的；例如`age = 18`，这里创建了一个变量`age`并赋值为18。不可以先定义变量再赋值。

````python
age = 18 # 变量名为age
name = "test"

sex # 先定义变量sex再赋值直接报错
sex = "男" 
````

#### 常量
常量是指在程序运行过程中值不能改变的变量。Python没有内置的常量类型，但可以通过命名约定来表示常量，通常使用全大写字母来命名常量；

常量的命名规范：全部大写，单词之间用下划线连接

````python
SYS_URL = "www.baidu.com"
````


### 注释

注释是用来解释代码的文本，Python中有两种类型的注释：
- 单行注释：以#开头，后面跟注释内容。
- 多行注释：使用三个单引号（'''）或三个双引号（"""）包裹注释内容。
- 文件编码注释：比较特殊，需要在文件的首行使用，指定当前文件的字符编码； 例如：# coding=utf-8（在 Python3 中，可以不写文件编码声明，默认就是utf-8）


````python
# 这是一个单行注释  

'''
这是一个多行注释
可以包含多行文本
''' 

"""
这也是一个多行注释
可以包含多行文本
""" 
````

````python
# coding=utf-8 
# 这是一个文件编码注释，指定当前文件的字符编码为utf-8
````

### 数据类型

每个数据都有对应的类型，类型决定了数据的属性和行为。

Python中常见的基本数据类型包括：
- 整数型（int）：表示整数。
- 字符串类型（str）：用于表示文本数据。通过单引号（'str'）或双引号（"str"）或三单引号/三双引号('''str''' / """str""")来定义字符串。
- 浮点型（float）：表示带有小数部分的数字。
- 布尔型（bool）：表示真（True）或假（False）的值。
- 其他类型后续会学到

type()函数可以用来查看变量的数据类型。

````python
age = 18 # 整数型
name = "test" # 字符串类型
name2 = 'test' # 字符串类型
name3 = '''test''' # 字符串类型
name4 = """test""" # 字符串类型
height = 1.75 # 浮点型
flag = True
print(type(age)) # <class 'int'>
print(type(name)) # <class 'str'>
print(type(height)) # <class 'float'>
````

注意事项：在print()打印的时候，整数和浮点数底层会自动转换为字符串类型进行输出;
如果整数打印时长度大约超过4300位会报错，这是python默认限制的长度，可以通过设置取消限制；


````python
num = 100**9999
print(num) # 报错
import sys
sys.set_int_max_str_digits(0) # 设置为0表示不作任何限制，只处理整数转字符串，与浮点数无关
print(num) # 不报错
````


#### 字符串的格式化输出

**字符串拼接**

1. 使用加号（+）进行字符串拼接，注意需要将非字符串类型转换为字符串类型；
2. 占位符（%）进行字符串拼接，使用%后面跟一个字符来表示占位符的类型，例如%s表示字符串，%d表示整数，%f表示浮点数等；占位符后面跟一个元组，元组中包含要替换占位符的值；
3. 使用f-string（格式化字符串字面量）进行字符串拼接，f-string允许在字符串中直接嵌入变量和表达式，使用花括号{}来包含变量或表达式，并且在字符串前加上字母f；

````python

# 使用加号（+）进行字符串拼接
name = "test"
age = 18
detail1 = "我的名字是" + name + "，今年" + str(age) + "岁" # 我的名字是test，今年18岁

# 使用占位符（%）进行字符串拼接
detail2 = "我的名字是%s，今年%d岁" % (name, age) # 我的名字是test，今年18岁

# 使用f-string进行字符串拼接
detail3 = f"我的名字是{name}，今年{age}岁" # 我的名字是test，今年18岁
````

**占位符精度控制**

在占位符前方，可以使用`m.n`的形式来指定精度
- 整数类型（%d）可以使用`%m.d`来指定最小宽度和精度，m表示最小宽度，如果实际宽度不足m，则会在左侧补空格；d表示整数部分的最小位数，如果实际位数不足d，则会在左侧补0；
- 浮点数类型（%f）可以使用`%m.nf`来指定最小宽度和小数位数，m表示最小宽度，如果实际宽度不足m，则会在左侧补空格；n表示小数部分的位数，如果实际位数不足n，则会在右侧补0；
- 字符串类型（%s）可以使用`%m.s`来指定最小宽度和最大字符数，m表示最小宽度，如果实际宽度不足m，则会在左侧补空格；s表示字符串的最大字符数，如果实际字符数超过s，则会截断字符串；
````python
pi = 3.141592653589793
# %m.nf表示保留n位小数，m表示总宽度（包括小数点和小数部分），如果实际宽度不足m，则会在左侧补空格
print("圆周率的值是：%.2f" % pi)
# 输出：圆周率的值是：3.14
# %m.d表示整数部分的最小位数，如果实际位数不足d，则会在左侧补0
num = 42
print("数字的值是：%05d" % num)
# 输出：数字的值是：00042
# %m.s表示字符串的最大字符数，如果实际字符数超过s，则会截断字符串
text = "Hello, World!"
print("文本内容是：%.5s" % text)
# 输出：文本内容是：Hello   
````

**转义字符**
转义字符是用来表示一些特殊字符的符号，常见的转义字符包括：
- \n：换行符
- \t：制表符（Tab）
- \\：反斜杠
- \'：单引号
- \"：双引号
- \r：回车符
- \b：退格符（删除前一个字符）
- \f：换页符
- \v：垂直制表符
- \0：空字符

````python
# 使用转义字符进行字符串输出
print("Hello\nWorld") # Hello换行World
print("Hello\tWorld") # Hello制表符World
print("Hello\\World") # Hello\World
print("She said, \"Hello, World!\"") # She said, "Hello,    World!"
````   


### 数据类型转换

使用内置函数进行数据类型转换，例如int()、float()、str()等；
- 如果转换失败会抛出异常，例如将一个非数字字符串转换为整数会抛出ValueError异常；
- 如果转换成功但存在精度损失，例如将一个浮点数转换为整数会丢失小数部分；将一个大整数转换为浮点数可能会丢失精度。
- 如果转换成功但存在范围限制，例如将一个超出整数范围的浮点数转换为整数会抛出OverflowError异常；将一个超出字符串长度限制的对象转换为字符串会抛出ValueError异常。


### 运算符

运算符是用来执行特定操作的符号或关键字，Python中常见的运算符包括：
- 算术运算符：+（加）、-（减）、*（乘）、/（除）、%（取模）、**（幂）、//（取整除）
- 比较运算符：==（等于）、!=（不等于）、>（大于）、<（小于）、>=（大于等于）、<=（小于等于）
- 逻辑运算符：and（与）、or（或）、not（非）
- 位运算符：&（按位与）、|（按位或）、^（按位异或）、~（按位取反）、<<（左移）、>>（右移）
- 赋值运算符：=（赋值）、+=（加赋值）、-=（减赋值）、*=（乘赋值）、/=（除赋值）、%=（取模赋值）、**=（幂赋值）、//=（取整除赋值）

**运算符优先级：括号 > 幂 > 乘除取模 > 加减 > 位运算 > 比较运算 > 逻辑运算 > 赋值运算**

### 输入语句

input()函数用于从用户输入获取数据，输入的数据默认是字符串类型，可以通过指定提示信息来引导用户输入；

````python
name = input("请输入你的名字：") # 请输入你的名字：test
age = input("请输入你的年龄：") # 请输入你的年龄：18
print(f"你的名字是{name}，你的年龄是{age}") # 你的名字是test，你的年龄是18
````
