## 面向对象



### 简单概述

- **对象**：对象是一个抽象的概念，任何东西都是对象，例如一辆车，一个人，一杯咖啡；且对象是一个具有属性和行为的个体；例如一个人，他有头发，有眼睛，这是他的属性，他要去上班，上班就是他的行为；
- **面向对象**：指的是一种思想。以对象为中心去组织代码，他更关注谁来做这件事
- **类（class）**：是用来描述一类事务的模板，他规定了一类事务的属性和行为；



- 实例：一个类的实例，也叫对象
- 实例化：要创建某个类的对象的过程



### 基本使用



#### 类的定义



**格式：**

````python
class 类名: # 类名通常采用大驼峰规则命名
    # 方法a（方法指的是函数，如果一个函数定义到了类中，他就叫方法）
    # 方法b
    
    # 初始化方法：__init__，主要用来给对象添加属性,当创建一个对象时，python会自动调用该方法进行初始化
    # 初始化方法接收到的参数为 当前正在创建的对象的实例，以及其他自定义参数
    # 且类必须定义 __init__ 初始化方法
    def __init__(self, 参数1，参数2 ...):
        # 通过self给当前实例添加属性，语法格式为：self.属性名 = 属性值
        self.属性名1 = 参数1
        self.属性名2 = 参数2
        
````



#### 创建实例



格式：

````python
实例名 = 类名(参数1，参数2 ...)	
````



1. 可通过 `实例名.属性名` 的方式去获取和修改属性
2. 通过`实例.__dict__` 可以查看实例的所有属性
3. 可以通过`实例.属性名 = 值` 来添加新的属性
4. 可以通过 `type()` 来查看实例的类型，或者说是那个类创建的



#### 自定义方法

将函数写到类中就是自定义方法，第一个参数也是`self`，调用方法时只需要通过`实例名.方法名()`进行调用

````python
class Person:
    def __init__(self, name):
        self.name = name
    def eat(self):
        print(self.name + "在吃饭") # self.属性名可以获取属性值
    def eat2(self, str):
        print(self.name + "在吃" + str)
person = Person("jock")
print(person.name)
person.eat()
person.eat2("辣椒炒肉拌面")
````



### 实例属性、类属性



- 实例属性：通过`实例.属性 = 值`的方式定义的属性，实例之间不互通
- 类属性：在类中方法外定义的属性，实例之间共享，可通过类访问也可通过实例访问



注意：不可以通过`实例.属性 = 值` 去修改类属性的值，这个操作只会被识别为创建/修改实例属性的值



````python
class Person:
    sex = 1 # 类属性
    def __init__(self, name):
        self.name = name # 实例属性

person = Person("jock")
person.age = 20 # 实例属性
person2 = Person("jock2")
person2.age = 200
Person.sex = 2
person.sex = 3 # 不会影响类属性

````



### 实例方法、类方法、静态方法



- 实例方法：定义在类中的方法就是实例方法，第一个参数是**实例本身**，通常定义为`self`,`self`参数可以获取实例属性。通过实例调用
- 类方法：使用`@classmethod`装饰器修饰，第一个参数是**类本身**，通常用形参`cls`接收，`cls`可以获取类属性。通过类调用（也可以通过实例调用，但不推荐）
- 静态方法：使用`@staticmethod`装饰器修饰，方法没有`self`或`cls`参数，只是单纯的定义在类中。可通过类名/实例调用，推荐通过类名调用



````python
class Person:
    max_age = 100
    address = "中国"
    
    def __init__(self, name, age, sex):
        self.name = name;
        self.age = age;
        self.sex = sex;
    
    # 实例方法
    def eat(self, food_name):
        print(self.name + "正在吃" + food_name)
    
    # 类方法
    # 通常用于定义操作类属性的方法
    @classmethod
    def update_address(cls, address):
        cls.address = address
    
    # 静态方法
    @staticmethod
    def mask_idcard(idcard):
        return idcard[:6] + '********' + idcard[-4:]
            
````



### 继承





一个类可以继承另一个类的属性和方法

作用：可以实现代码复用和扩展

继承的类叫子类，被继承的类叫父类



**语法：**

格式：`class 类名 (继承的类):`

在子类的初始化方法中，可以通过两种方式初始化父类

1. `super().__init__(属性)` 推荐
2. `父类名.__init__(属性)` 一般多重继承场景使用



**方法重写**

如果子类定义了和父类一样的方法，那么就是方法重写，在调用方法时调用的就是子类的方法，而不是父类的（父类的方法会被覆盖）



**常用方法**

| 函数                     | 作用                                       |
| ------------------------ | ------------------------------------------ |
| `isinstance(obj, Class)` | 判断对象是否为**指定类**或其**子类**的实例 |
| `issubclass(Sub, Super)` | 判断一个类是否是另一个类的**子类**         |



**多重继承**

可以继承多个父类，用逗号分隔，从而拥有多个父类的属性和方法





````python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def eat(self):
        print("吃")
class Student(Person): # 继承
    def __init__(self, name, age, sex):
        super().__init__(name, age)
        self.sex = sex
        
    def eat(self): # 重写
        print("吃粑粑")
student = Student("John", 22, "Male")
print(student.name)
student.eat()

# 类的__mro__属性：用于记录属性和方法的查找顺序
# 通过实例去查找属性或方法时，会现在实例自身上寻找，如果没有，就按照__mro__中所记录的顺序去查找
print(Student.__mro__)
````
