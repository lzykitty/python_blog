## 流程控制

### 分支

通过条件来决定走那个分支，为true则进去，否则不进去

- 先走if语句，如果条件满足就执行对应的代码块；
- 如果不满足就继续往下走elif语句；
- 直到找到满足条件的语句或者走到else语句为止。

语法：
````python

# 单分支
if 条件:
    代码块

# 双分支
if 条件:
    代码块1
else:
    代码块2

# 多分支
if 条件1:
    代码块1
elif 条件2:
    代码块2
elif 条件3:
    代码块3
else:
    代码块4
````

案例
````python
age = 18    
if age >= 18:
    print("你已经成年了")
else:
    print("你还未成年")
````


### 循环

让代码重复执行，直到满足某个条件为止

- for循环：用于遍历序列（如列表、元组、字符串等）或可迭代对象，语法为`for 变量 in 序列:`，每次循环会将序列中的一个元素赋值给变量，并执行代码块。
- while循环：用于在满足某个条件的情况下重复执行代码块，语法为`while 条件:`，每次循环会先判断条件是否为真，如果为真则执行代码块，否则跳出循环。
- break语句：用于提前退出循环，无论循环条件是否满足。
- continue语句：用于跳过当前循环的剩余代码，直接进入下一次循环的判断。
- else语句：可以与for循环或while循环配合使用，当循环正常结束（没有被break语句打断）时，会执行else语句块中的代码。

语法
````python

# for语法
for 变量 in 序列: # 序列可以是列表、元组、字符串等可迭代对象
    代码块
else:
    代码块

# while语法
while 条件: # True or False
    代码块
else:
    代码块
````


案例
````python

# for循环配合else使用
for i in range(5): # range(5)会生成一个包含0到4的整数序列
    print(i)
else:
    print("循环结束")

# while循环配合else使用
count = 0
while count < 5:
    print(count)
    count += 1
else:
    print("循环结束")

# break使用
while count < 5:
    if count == 3:
        break
    print(count)
    count += 1

# continue使用
while count < 5:
    count += 1
    if count == 3:
        continue
    print(count)

````
